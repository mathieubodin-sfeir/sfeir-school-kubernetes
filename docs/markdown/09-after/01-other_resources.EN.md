# What's next? > **Other resources**

Throughout this training, we have covered a set of Kubernetes resources. To go further, you can look into:
* **statefulSet:** Manages a set of pods guaranteeing name and storage uniqueness (useful for databases for example).
* **daemonSet:** Forces pods to be distributed across all cluster nodes (useful for logging and monitoring agents, ...).
* **jobs:** Allows launching a one-off action (prepare a database, ...), can be in cron mode (backups).
* **storageClass:** Manages different storage types.
